# Mkart
It is an e-commerce MERN full stack application.

# URL
https://mkart1.netlify.app

# Features
• View Products and product details,Add to Cart, Add Reviews for products, Login and Sign up.  
• Monolith Arcitecture.  
• Responsive UI.  

# Technologies Used
1 - ReactJS  
2 - ExpressJs  
3 - NodeJs  
4 - MongoDB  
5 - Redux
